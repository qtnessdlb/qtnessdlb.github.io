Preparation

Member diminta untuk mengirimkan wallpaper "unik" sesuai dengan quest yang diberikan sebagai penilaian. Wallpaper yang "unik" akan bernilai 5 poin, sementara wallpaper yang sama persis dengan member lain, akan mendapatkan 1 poin. (Wallpaper dengan resolusi diatas 3840 x 2160 akan mendapatkan tambahhan 10 poin per wallpaper, & Jika terdapat 10 wallpaper dengan resolusi tinggi, akan dijadikan pack premium, dengan tambahan 50 poin).

Bagaimana cara untuk menjadikan ZIP / RAR pada sebuah file?, silahkan download terlebih dahulu software yang dibutuhkan, pada perangkat, kemudian lakukan tahap berikut:

Download RAR ( RAR )

  1. Tutorial RAR PC

  2. 1. Jika menggunakan PC, pilih terlebih dahulu semua file yang ingin di kompress,

  3. 2. Jika sudah, lakukan klik kanan dimana saja untuk memunculkan opsi,

  4. 3. Kemudian pilih "Add to Archive" dan pilih "ZIP" jika aplikasi sudah terbuka

  5. Note: Download dan Install dulu aplikasi nya ya :D
